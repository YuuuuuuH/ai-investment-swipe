# 项目使用说明

## 快速上手

1. `cd backend`
2. `node init_db.js`
3. `node index.js`（默认使用 `demo_projects_20.csv` 初始化数据库）
4. `node index.js`
5. 在微信小程序开发者工具里编译并运行
6. 输入用户 ID 与密码登录

## 功能说明

### 投资经理
- **卡片流**：左右滑动提交决策，点击卡片翻转查看同行评价
- **备注**：提交决策时可附带文字意见
- **历史**：在看板中回溯自己处理过的所有项目细节

### 管理员
- **进度看板**：展示每个经理的批阅进度
- **数据导入**：直接上传 CSV 文件，系统自动对比 Hash 执行增量更新
- **全局视角**：查看全员反馈，对比不同人的评价差异
